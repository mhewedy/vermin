exec

To execute remote commands against the VM without logging into it, use the exec subcommand.

For example, you might need to run a quick command and show the result, e.g. showing the result of a specific configuration file.

The following example using the cat command to show the network configurations in an ubuntu/focal image: 

$ vermin exec vm_01 cat /etc/netplan/00-installer-config.yaml
# This is the network config written by 'subiquity'
network:
  ethernets:
    enp0s3:
      dhcp4: true
  version: 2

Another example shows how to download a package (nginx) wihtout logging into the VM:

$ vermin exec vm_01 -- sudo apt install nginx -y

WARNING: apt does not have a stable CLI interface. Use with caution in scripts.
