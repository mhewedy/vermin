gui

gui subcommand used open GUI for the VM. The GUI is based on Virtualbox console.

You will make most use of it if you are installing a desktop on top of one of the VMs you have. This medium post shows how to install ubuntu mate desktop on top of ubuntu/focal. 

$ vermin gui vm_07

You will use the default password (vermin) to login to the GUI console

After finish you can close the window while keeping the VM running so to continue use it from the command line.
