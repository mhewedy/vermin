tag

tag subcommand used to add or remove tag to a VM. You can tag a VM as many times as you want.

Tags are one of the fields that can be used in filters of the command.

To add a tag to a VM: 
$ vermin tag vm_01 elk
You can add another tag to the same VM: 
$ vermin tag vm_01 elasticsearch

Both tags will be listed in the : 

$ vermin ps
VM NAME      IMAGE          CPUS      MEM       DISK         TAGS
vm_01        ubuntu/focal64   1         1024      2.7GB        elasticsearch, elk

Ta remove a tag from a VM: 

$ vermin tag -r vm_01 elk
